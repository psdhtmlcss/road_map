2. Какие типы в JS изменяемы, какие нет?

Все примитивные типы в JS являются неизменяемыми, это:

  1. String
  2. Number
  3. Boolean
  4. Symbol
  5. Bigint

Сложные типы (объекты, массивы, функции) являются изменяемыми.

3. Что значит ссылка на переменную?

Пример: создадим переменную a и присвоим ей значение в виде массива [1, 2, 3]. Затем создадим переменную b и присвоим ей значение a. Изменим первое значение в массиве b[0] = 30;. Если выведем в консоль console.log(a[0]);, то увидим, что и здесь первое значение массива изменилось. Т.е. переменная b хранит не копию массива a, а ссылку на него.

4. Что такое NaN? Как проверить, что переменная NaN?

NaN — это результат некоторых математических операций в JS, означающий not a number (не число).

Мы получим NaN, если:

  1. попытаемся произвести арифметическую операцию с операндом, не являющимся числом (кроме сложения) например, 'number' * 5;
  2. операнды не определены, например, undefined + undefined;
  3. при ошибке парсинга, например, parseInt('{}');
  4. один из операндов в математической операции равен NaN, например, NaN + 56;
  5. произведем арифметическую операцию, результат которой не входит в диапазон действительных чисел.

При помощи метода Number.isNaN(someValue) можно узнать равно ли переданное значение NaN.

5. Что значит создавать переменную через конструктор или через литерал?

Создавать переменную через конструктор означает использовать специальную функцию-конструктор и оператор new, например, const a = new String('str'); — создание переменной, содержащей строку, через конструктор.

Создавать переменную через литерал, означает использовать упрощенный способ, т.е. записывать значение без функции-конструктора, const a = 'str';
