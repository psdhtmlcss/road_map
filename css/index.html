Вопросы по CSS уровня Junior 1

1. Какие значения бывают у свойства display? В чем их особенность?

Значений для свойства display много, чаще всего встречаются:

  • block — делает элемент блочным;
  • inline-block — делает элемент строчным, но позволяет задать значения ширины и высоты;
  • flex — делает элемент флекс-контейнером;
  • grid — делает элемент грид-контейнером;
  • ...

Другие значения свойства display на developer.mozilla.org

2. Что делает свойство margin?

Свойство margin задает внешние отступы у элемента.
