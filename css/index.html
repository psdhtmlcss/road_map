Вопросы по CSS уровня Junior 1

1. Какие значения бывают у свойства display? В чем их особенность?

Значений для свойства display много, чаще всего встречаются:

  • block — делает элемент блочным;
  • inline-block — делает элемент строчным, но позволяет задать значения ширины и высоты;
  • flex — делает элемент флекс-контейнером;
  • grid — делает элемент грид-контейнером;
  • none — скрывает элемент;
  • ...

2. Что делает свойство margin?

Свойство margin задает внешние отступы у элемента.

Свойство, которым можно отодвинуть элемент от соседей. Или придвинуть. | DOKA.GUIDE

3. Какие есть особенности у отрицательных значений свойства margin?

Отрицательные значения margin-top/left сдвигают элемент вверх/влево. Остальные элементы это учитывают в отличие от сдвига через position.

Отрицательные значения margin-right/bottom заставляют другие элементы думать, что блок меньше по размеру справа/снизу, чем он есть на самом деле.

Margin | LEARN.JAVASRIPT.RU

4. Что такое margin collapsing?

Это схлопывание соприкасающихся по вертикали отступов у элементов. Сработает отступ, который больше.
