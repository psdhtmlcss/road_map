Вопросы по HTML уровня Junior 1

1. Как сделать верстку семантической и зачем это нужно?

Семантика (гр. "обозначающий"). В HTML смысловое использование тегов, т.е. с учетом их предназначения. Это нужно для браузеров и различных технических устройств, например, скринридеров для указания значения и роли того или иного элемента на странице.

Как работает семантика и зачем вообще она нужна | DOKA.GUIDE

2. Что такое Doctype и зачем он нужен?

Doctype — тип документа, указывающий какой стандарт или версия языка используется в данном документе. В настоящее время используется запись <!doctype html>

Doctype | DOKA.GUIDE

3. Какие есть отличия у тегов div, p, span, a

Теги div и p являются блочными. Теги span, a — строчными. Каждый из этих тегов имеет свое предназначение: div — обозначение раздела, блока; p — абзац; span — выделение части строки; a — тег для создания ссылки.

4. Что такое инлайновые элементы?

Инлайновые или строчные элементы — это элементы, которые занимают столько места, сколько необходимо для их обозначения. Для таких элементов нельзя задать свойства ширины и высоты.

5. Как создавать таблицы в HTML?

Создавать таблицы в HTML можно, используя специальные теги: table, thead, tbody, tfoot и другие. А также стили для стилизации таблицы и ее элементов.

Определение некоторых свойств:

  • border-spacing — свойство CSS задает отступы между рамками ячеек;
  • border-collapse — свойство CSS управляет отображением рамок ячеек таблицы;
  • table-layout — свойство CSS определяет как браузер должен вычислять ширину ячеек таблицы;
  • caption — тег нужен для указания подписи к таблице;
  • scope — html-атрибут, связывающий между собой ячейки с заголовком и обычные ячейки. Предназначен для скринридеров.

6. В чем заключается отличие чекбоксов от радиокнопок?

  • Использование чекбоксов позволяет выбрать несколько вариантов из списка.
  • У одиночного чекбокса можно снять галочку, у радиокнопки нет, только сбросив форму.

7. Как связать радиокнопки в группу?

Чтобы связать радиокнопки в группу, необходимо задать каждой одинаковое значение для атрибута name.

8. БЭМ

Что такое БЭМ?

БЭМ (блок, элемент, модификатор) — компонентный подход в веб-разработке, в основе которого лежит принцип разделения интерфейса на независимые блоки.

БЭМ | ru.bem.info

Что такое Блок?

Блок — функционально независимый компонент страницы, который может быть неоднократно использован. В HTML блоки представлены атрибутом class. Название блока характеризует смысл, а не состояние. Блоку не задают внешнюю геометрию и позиционирование. Блоки можно вкладывать друг в друга.

Блок | ru.bem.info

Может ли блок содержать в себе другие блоки?

Может.

Почему не следует задавать внешнюю геометрию блокам?

Блоки могут быть повторно использованы в других частях страницы, поэтому не должны влиять на соседнее окружение.

Что такое элемент?

Элемент — составная часть блока, которая не может использоваться в отрыве от него.

Структура полного имени соответствует схеме: блок__элемент.

Элемент | ru.bem.info

Может ли элемент содержать другие элементы?

Может.

Может ли элемент содержать блоки (блок, к которому он относится или другие блоки)?

Элемент может содержать другие блоки. У элемента может быть только один родитель — блок, к которому он принадлежит.

Можно ли создавать элемент элемента?

Нет.

Может ли элемент использоваться вне блока?

Нет.

Может ли существовать блок без элементов?

Может.
